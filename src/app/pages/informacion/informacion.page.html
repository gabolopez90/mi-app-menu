Información Programación III / IONIC Framework de Desarrollo Híbrido

Ionic es un SDK completo de código abierto para el desarrollo de aplicaciones móviles híbridas, creado originalmente por Max Lynch, Ben Sperry y Adam Bradley de Drifty Co. en 2013.

Ventajas principales:

Un solo código para iOS, Android y Web. Basado en tecnologías estándar (HTML, CSS, JS). Amplia biblioteca de componentes UI pre-diseñados. Integración sencilla con Angular, React o Vue.
